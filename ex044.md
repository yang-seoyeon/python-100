# 문제44 : 각 자리수의 합

## 사용자가 입력한 양의 정수의 각 자리수의 합을 구하는 프로그램을 만들어주세요

**예를들어**
### 18234 = 1 + 8 + 2 +3 + 4이고 정답은 18입니다.
### 3849 = 3 + 8 + 4 + 9이고 정답은 24입니다.

#### >>입력
1234
#### >>출력
10

------------------

n = input('양의 정수를 입력하세요: ')
list = []
score = 0
for i in range(1, len(n) + 1, 1):
    list.append(n[i - 1 : i])
    s = int(list[i - 1])
    score += s
print(score)

# 문제46 : str 자료형의 응용

## 1부터 20까지의(20을 포함) 모든 숫자를 일렬로 놓고 모든 자릿수의 총 합을 구하세요.
## 예를 들어 10부터 15까지의 모든 숫자를 일렬로 놓으면 101112113141415이고 각 자리의 숫자를 더하면 25입니다.


------------

twenty = '1234567891011121314151617181920'
score = 0
for i in range(len(twenty)):
    t = twenty[i - 1]
    score += int(t)

print(score)
